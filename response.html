node js 서버 구현해줘

먼저 Node.js를 설치해야 합니다. Node.js 공식 홈페이지에서 다운로드 받을 수 있습니다.

설치가 완료되면, 다음과 같이 간단한 서버를 구현할 수 있습니다.

const http = require('http');

const hostname = '127.0.0.1';
const port = 3000;

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello World\n');
});

server.listen(port, hostname, () => {
  console.log(`Server running at http://${hostname}:${port}/`);
});

이 코드는 간단한 HTTP 서버를 생성하고, 요청이 들어오면 "Hello World"를 응답으로 보내줍니다. 이 서버를 실행하려면, 터미널에서 다음 명령어를 입력합니다.

node server.js

이제 브라우저에서 http://127.0.0.1:3000에 접속하면 "Hello World"가 출력됩니다.
